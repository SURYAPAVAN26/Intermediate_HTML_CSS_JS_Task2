# Intermediate_HTML_CSS_JS_Task2
# 🌐 Intermediate HTML, CSS & JavaScript Project

## 📖 Project Description

This project was developed as part of the **Intermediate HTML, CSS & JavaScript Task**.

The application combines multiple web development concepts into a single responsive website, including:

- Contact Form Creation
- JavaScript Form Validation
- Responsive Layout using Flexbox and CSS Grid
- Dynamic To-Do List
- DOM Manipulation
- Mobile-Friendly Design

---

## ✨ Features

### 📬 Contact Form
- User-friendly contact form
- Name, Email, and Message fields
- Input validation using JavaScript
- Success message displayed after submission

### 📱 Responsive Design
- Responsive navigation bar using Flexbox
- Responsive content layout using CSS Grid
- Media Queries for mobile and tablet screens

### ✅ Dynamic To-Do List
- Add tasks dynamically
- Remove tasks instantly
- Real-time DOM manipulation

---

## 🛠 Technologies Used

- HTML5
- CSS3
